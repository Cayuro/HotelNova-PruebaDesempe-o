# HotelNova - Indice de Lectura Rapida

Este archivo te sirve como punto de entrada para entender el proyecto sin leer todo de golpe.

## Orden recomendado de lectura
1. [MODELS_NOTEBOOK.md](MODELS_NOTEBOOK.md)
2. [DAOS_NOTEBOOK.md](DAOS_NOTEBOOK.md)
3. [CONTROLLERS_NOTEBOOK.md](CONTROLLERS_NOTEBOOK.md)
4. [TEMPLATES_HOTELNOVA.md](TEMPLATES_HOTELNOVA.md)
5. [SPEC_DRIVEN_HOTELNOVA.md](SPEC_DRIVEN_HOTELNOVA.md)

## Que vas a encontrar aqui
- MODELS_NOTEBOOK: que representa cada entidad y para que sirve.
- DAOS_NOTEBOOK: que pregunta responde cada DAO y que consultas especiales tiene.
- CONTROLLERS_NOTEBOOK: como fluye cada caso de uso desde la vista hasta el DAO.
- TEMPLATES_HOTELNOVA: plantilla mental para aprender y luego implementar.
- SPEC_DRIVEN_HOTELNOVA: reglas, contratos y plan general del sistema.

## Regla de uso
- Si algo no se entiende, volver al indice y leer solo la capa relacionada.
- No hace falta memorizar todo; hace falta entender el papel de cada capa.
